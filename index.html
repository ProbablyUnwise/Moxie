What fresh hell is this?!

This is Moxie, my project name for what will eventually be the new theme for BattleMaster. Moxie is a responsive, mobile-first theme that will bring the game's UI into the current generation and make the user experiance smoother as a whole, especially for those who like to read their messages throughout the day on their phones! But just because Moxie is mobile-first, doesn't mean that it isn't for desktop users. Moxie will work equally well on both desktop and mobile devices such as tablets and phones, and will adjust herself to fit cleanly on both. Only the UI elements for your current platform will be displayed, so you can always be sure the messages you need to read each turn, and the actions you need to take, will be easy to find and make use of. Moxie is smart like that!

You can expect Moxie to be ready sometime in 2021. Until then, feel free to visit here every so often to see the current progress! This is the test server for the theme, so it always shows the latest version I think is ready for public consumption. To see how Moxie works, try visiting this site on your mobile device, or resize your browser window to watch it in action!
